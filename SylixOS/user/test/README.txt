测试说明:

0. bspk210 默认使用的 BASE 版本为: sylixos-base-lite 裁掉了网络部分、打开了SD支持。

1. bspk210 中本身包含 AI 模块的测试程序 和 liblvgl 库的测试程序。

2. spi-flash 和 spi-sdcard 存在公用引脚，所以只能单独使用其中一个；可以通过 configk210.h 中的宏来选择是使用 flash 还是 sdcard。

3. 本目录下包含对应驱动的测试程序，部分驱动需要使用杜邦线连接相关引脚后，在使用测试程序进行测试；具体请看测试程序中的注释说明。

4. 由于该板卡的内存比较小，建议测试驱动时，对单个驱动单独进行测试。

5. KPU 测试和 (SDcard 测试 + VideoFW 测试) 要分开，因为 SDcard测试 需要1M内存，当包含KPU驱动时是没有1M内存的，所以会报SDcard挂载失败。
